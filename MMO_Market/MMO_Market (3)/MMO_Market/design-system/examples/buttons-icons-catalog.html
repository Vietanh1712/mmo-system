Buttons & Icons Catalog

Tổng hợp toàn bộ button và icon đang có CSS trong design system.

Button variants

Base class bắt buộc: .ds-btn. Variant class chọn theo ngữ cảnh hành động.

Button states

Anchor button

Dùng `disabled` cho button thật, dùng `.ds-disabled` khi element không hỗ trợ disabled như link.

Buttons with icons

Icon glyphs

Sort arrows ds-icon-sort
Cloud Upload ds-icon-cloud-upload
Flag ds-icon-flag
Notification Bell ds-icon-bell

Icon dùng `currentColor`, nên đổi màu bằng `color` của chính icon hoặc phần tử cha.

Icon buttons

Button chỉ có icon cần `aria-label` để hỗ trợ accessibility.
